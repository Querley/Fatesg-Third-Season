# Lista 3 de Matrizes em Java - N1

Este projeto contém diversos exercícios em Java que exploram o uso de **matrizes (arrays bidimensionais)** e práticas básicas de **estruturação e manipulação de dados**.
Os códigos foram feitos de forma **simples e modular**, com foco em clareza e prática dos conceitos fundamentais de **loops aninhados** e **armazenamento em 2D**.

## Exercícios

1. **Matriz 3x3 com valores de 1 a 9**
   Preenche uma matriz 3x3 com números em ordem crescente de 1 a 9 e imprime o resultado.

2. **Matriz 4x4 com números pares**
   Cria uma matriz 4x4 preenchida apenas com números pares, começando do 2.

3. **Matriz 5x5 com valores aleatórios**
   Gera uma matriz 5x5 com números aleatórios entre 0 e 100.

4. **Média de notas de alunos (3x3)**
   Lê as notas de 3 alunos com 3 provas cada, calcula e exibe a média de cada aluno.

5. **Tabuleiro de xadrez 8x8**
   Cria uma matriz 8x8 preenchida alternando os caracteres `'B'` e `'W'`, simulando um tabuleiro de xadrez.

---

## Observações

* Todos os exercícios utilizam **estruturas de repetição aninhadas** para manipulação de matrizes.
* As soluções foram feitas no formato **imperativo**, priorizando a clareza e o entendimento do funcionamento das matrizes.
* Cada exercício é independente, podendo ser executado separadamente.
