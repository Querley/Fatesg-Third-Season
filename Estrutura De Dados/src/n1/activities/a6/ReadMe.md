# Lista 6 de Pilhas em Java - N1

Este projeto contém exercícios em Java voltados à prática de **pilhas (Stack)** e ao entendimento do princípio **LIFO (Last-In, First-Out)**.
Cada programa foi feito de forma **simples e objetiva**, aplicando a pilha em diferentes contextos e resolvendo problemas clássicos de estrutura de dados.

## Exercícios

1. **Verificador de Palíndromo**
   Lê uma palavra ou frase e usa uma pilha para verificar se é um palíndromo (lê-se igual de trás para frente).

2. **Verificação de Símbolos Balanceados**
   Lê uma expressão contendo parênteses, colchetes e chaves e utiliza uma pilha para verificar se os símbolos estão corretamente balanceados.

3. **Conversão de Infixo para Pós-fixo**
   Converte uma expressão matemática da forma infixa (ex: `2 + 3 * 4`) para a forma pós-fixa (`2 3 4 * +`) usando uma pilha para controle de operadores.

4. **Inversão de Fila com Pilha**
   Inverte os elementos de uma fila (`Queue`) usando uma pilha (`Stack`), demonstrando a relação entre as duas estruturas.

---

## Observações

* Todos os exercícios utilizam **Stack** e **Queue** da biblioteca padrão Java.
* As soluções seguem o estilo **imperativo**, com código direto e fácil de entender.
* Cada exercício foi estruturado em uma classe independente dentro do mesmo arquivo para facilitar a execução individual.
